# big_hex_transport
Cleverly combine hexadecimal numbers to create the largest possible number using limited components.


## Build:

**Build for web:**
```bash
cd big-hex-transport
emcmake cmake -B build
cmake --build build

# run
emrun build/big-hex-transport.html
```

**Build nativ:**
```bash
cd big-hex-transport
cmake -B build-native
cmake --build build-native

# run
./build-native/big-hex-transport
```
